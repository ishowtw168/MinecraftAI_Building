# Contributing to Minecraft AI Studio

Thank you for your interest in contributing!

## Our Goal

Minecraft AI Studio aims to help creators design better Minecraft worlds through AI-assisted collaboration.

We welcome ideas, discussions, documentation improvements, templates, and technical contributions.

---

## Before You Contribute

Please:

- Read the project vision.
- Read the design principles.
- Check existing issues before creating a new one.

---

## Contribution Types

- Documentation
- AI Studio Members
- Templates
- Examples
- Minecraft Knowledge Base
- Technical Integrations
- Bug Reports
- Feature Requests

---

## Design Philosophy

Every contribution should follow these principles:

- Human-centered creation
- Clear reasoning
- Reusable knowledge
- Consistent design
- Modular architecture

---

## Pull Requests

Please explain:

- What problem this solves.
- Why the change is useful.
- Any design trade-offs.

---

Thank you for helping build Minecraft AI Studio!
