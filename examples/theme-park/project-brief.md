# Project Brief

## Project Name

Sky Isles Theme Park

---

## Project Vision

Create a Minecraft theme park built across two floating islands.

Each island should provide a completely different atmosphere while feeling like part of the same park.

The experience should encourage exploration, discovery, and memorable moments.

---

# Target Audience

Players who enjoy:

- Exploration
- Beautiful scenery
- Theme park attractions
- Adventure

---

# Core Design Goals

- Every island has a unique identity.
- Navigation is intuitive.
- Landmarks guide players naturally.
- Attractions encourage curiosity.
- The park feels alive.

---

# Theme Islands

## Island One

Fantasy Kingdom

Features:

- Castle
- Village
- Giant Tree
- Fantasy Coaster

---

## Island Two

Ancient Desert

Features:

- Pyramid
- Oasis
- Tomb Ride
- Desert Temple

---

# Constraints

- Vanilla Minecraft
- Survival-friendly scale
- Focus on visual storytelling
- Modular expansion for future islands

---

# Success Criteria

Players should:

- Never feel lost.
- Always notice something interesting.
- Easily recognize each island.
- Want to explore further.
