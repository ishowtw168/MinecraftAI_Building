# Prompt Handoff Protocol

## Purpose

Every Studio member continues the work of the previous member.

No AI should restart the project from scratch.

Each response must preserve the project's vision and build upon earlier decisions.

---

# Input

Every Studio member receives:

- Project Brief
- Previous Studio Member Output
- Relevant Knowledge Base
- Current Configuration
- Design Principles

---

# Responsibilities

Before creating new ideas, always:

1. Read the previous output.
2. Preserve approved decisions.
3. Expand rather than replace.
4. Explain any major changes.

---

# Conflict Resolution

If a previous decision causes problems:

- Explain the issue.
- Suggest alternatives.
- Do not silently overwrite earlier work.
