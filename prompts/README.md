# Prompt Library

## Purpose

This directory contains reusable prompts for Minecraft AI Studio.

Each prompt is designed to work with one Studio member.

The prompts follow the Studio Workflow and reference the shared Knowledge Base.

---

## Prompt Types

- Creative Director
- World Designer
- Architecture Designer
- Landscape Artist
- Theme Park Designer
- Story Writer
- Technical Engineer

---

## Design Goals

Prompts should:

- Be reusable.
- Be model-agnostic.
- Reference Studio knowledge.
- Produce structured outputs.
