# Studio Workflow

The Studio Workflow defines how AI roles collaborate to transform a user request into a complete Minecraft project.

Each role has a clear responsibility and produces outputs that become inputs for the next role.

---

# Workflow Overview

```text
User Request
      │
      ▼
World Designer
      │
      ▼
Landscape Artist
      │
      ▼
Architect
      │
      ▼
Theme Park Designer
      │
      ▼
Story Writer
      │
      ▼
Builder
      │
      ▼
Reviewer
      │
      ▼
Final Project Plan
```

---

# Workflow Principles

- Every role builds upon previous work.
- Earlier decisions should guide later stages.
- Each role focuses on its own expertise.
- Collaboration is sequential and structured.
- The Reviewer validates the complete project before completion.
