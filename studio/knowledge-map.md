# Knowledge Map

This document defines which knowledge resources each Studio role should reference during the world design process.

The purpose of this document is to establish a clear relationship between the AI Studio roles and the Knowledge Base, ensuring that every role works with the appropriate design knowledge.

---

# Overview

| Studio Role | Primary Knowledge | Status |
|--------------|-------------------|--------|
| Creative Director | Design Principles, Visual Hierarchy | ✅ |
| World Designer | Landmarks, Visual Hierarchy | ✅ |
| Architecture Designer | Fantasy Architecture | ✅ |
| Landscape Artist | Landscape Knowledge *(Future)* | 🚧 |
| Theme Park Designer | Theme Park Knowledge *(Future)* | 🚧 |
| Story Writer | Storytelling Knowledge *(Future)* | 🚧 |
| Technical Engineer | Workflow, Templates, Architecture | ✅ |
